# WPM

## Getting Started

1. Clone the repo
2. Install dependencies with your package manager
   - We use Bun
3. Run development server
   ```bash
   bun dev
   npm run dev
   yarn run dev
   pnpm run dev
   ```
4. Run convex server for leaderboard
   ```
   bunx convex dev
   ```
